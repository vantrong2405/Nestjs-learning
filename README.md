### Các bước cần làm để chạy dự án NestJS

#### 1. Cài đặt thư viện với câu lệnh: npm i
#### 2. Chạy dự án với câu lệnh: npm run dev
#### 3. Cài đặt Nest CLI : npm i -g @nestjs/cli@9.4.2
#### 4. Cài dặt package : nest g resource users (CRUD)
